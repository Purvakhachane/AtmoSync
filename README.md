🌤️ AtmoSync

Real-time weather intelligence with dynamic visuals and location-based news.


📌 Overview

Atmosync is a modern web application that combines real-time weather data with location-based news into a single, visually engaging dashboard.

It not only provides accurate atmospheric conditions like temperature, humidity, and wind speed, but also enhances the experience with dynamic animations such as sun, rain, clouds, and snow based on live weather conditions.

...

🚀 Features

🌤️ Real-time weather data (temperature, humidity, wind, condition)

🎨 Dynamic weather animations (sunny, rainy, snowy, cloudy)

📰 Location-based news (headline + description)

📍 Geolocation support (auto-detect user location).

💎 Modern glassmorphism UI.

⚡ Fast API integration with proper error handling.

...

🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript (ES6)

APIs Used:
Weather Data: WeatherAPI
News Data: GNews API
